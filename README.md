# POC Upload
[![Symfony](https://github.com/damienlagae/poc-upload/actions/workflows/symfony.yml/badge.svg)](https://github.com/damienlagae/poc-upload/actions/workflows/symfony.yml)

POC of file upload in Symfony with EasyAdmin and Azure Blob Storage for file management.

## Installation

```bash
make install
```

## Usage

```bash
make run
```
Go to [http://localhost:8000](http://localhost:8000)
